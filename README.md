# 🍅 番茄钟桌面应用

一个简洁、美观的 Pomodoro 番茄钟桌面应用，基于 Python `tkinter` 开发，帮助你高效管理专注时间与休息节奏。

---

## ✨ 功能特性

- **经典番茄工作法**：默认 25 分钟专注 + 5 分钟短休息 + 15 分钟长休息
- **可视化进度条**：圆形进度环实时显示剩余时间，直观清晰
- **自动循环提醒**：每完成 4 个番茄后自动进入长休息，劳逸结合
- **置顶窗口**：支持窗口始终置顶，方便随时查看剩余时间
- **系统提示音**：计时结束时播放蜂鸣提示音（Windows）
- **精致暗色主题**：采用 Catppuccin Mocha 配色，护眼舒适

---

## 🖼️ 界面预览

应用主界面包含：

- 顶部状态标签（🍅 专注时间 / ☕ 短休息 / 🌴 长休息）
- 中央圆形进度条与倒计时显示
- 操作按钮区：开始 / 暂停 / 重置
- 底部：完成番茄计数与置顶开关

---

## 🛠️ 技术栈

| 技术 | 说明 |
|------|------|
| Python 3 | 核心语言 |
| tkinter | GUI 界面框架 |
| threading | 后台计时线程 |
| winsound | Windows 系统提示音 |

> **注意**：本应用目前仅在 Windows 平台测试，`winsound` 模块为 Windows 专属。

## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/ly99LLL/tomatobinggo.git
cd tomatobinggo
```

### 2. 运行应用

```bash
python pomodoro.py
```

在 Windows 上推荐以无控制台方式启动：

```bash
pythonw pomodoro.py
```

> 确保你的 Python 环境已安装 `tkinter`（通常 Python 官方安装包已内置）。

---

## 📖 使用说明

1. 点击 **「开始」** 启动当前计时阶段（默认从专注时间开始）。
2. 计时过程中可随时 **「暂停」** / **「继续」**。
3. 点击 **「重置」** 将恢复初始状态，清空已完成计数。
4. 专注时间结束后自动进入休息阶段，并播放提示音。
5. 勾选/取消 **「窗口置顶」** 控制应用是否始终显示在最前。

---

## ⚙️ 自定义配置

如需调整时间参数，可直接编辑 `pomodoro.py` 中的类常量：

```python
class PomodoroTimer:
    WORK_MIN = 25          # 专注时长（分钟）
    SHORT_BREAK_MIN = 5    # 短休息时长（分钟）
    LONG_BREAK_MIN = 15    # 长休息时长（分钟）
    SESSIONS_BEFORE_LONG = 4  # 几个番茄后进入长休息
```
